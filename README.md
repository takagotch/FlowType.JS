### FlowType.JS
---
https://github.com/simplefocus/FlowType.JS

```js
$('body').flowtype({
  minimum : 500,
  maximum : 1200
});

$('body').flowtype({
  minFont : 12,
  maxFont : 40
});

$('body').flowtype({
  fontRatio : 30
});

```
